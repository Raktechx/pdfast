# PDFast MVP

An original, responsive browser-based PDF tools dashboard. The first version processes files locally in the browser, so no document backend is required.

## Included tools

- Merge multiple PDFs
- Extract selected PDF pages
- Convert JPG or PNG images into a PDF
- Export PDF pages to PNG images (ZIP download for multi-page PDFs)
- Rotate all pages in a PDF

## Run it

Open `index.html` in a modern browser while connected to the internet. The page loads PDF processing libraries from reputable public CDNs; your selected documents are not uploaded to those services.

For deployment, upload these three files to any static web host (for example, Cloudflare Pages, Netlify, or GitHub Pages).

## Revenue setup after launch

1. Publish a privacy policy and terms page for your real domain.
2. Connect web analytics and track completed jobs per tool.
3. Apply for Google AdSense after the site has useful pages and real visitors.
4. Start with a single clearly-separated display ad on the tool list or a completed-result screen—never beside a download button or during processing.
5. Offer a paid no-ads plan with batch work, larger files, and advanced server-side conversions.

Do not add live ad IDs until the site and privacy disclosures are ready. Use test ad IDs during any development or QA work.
